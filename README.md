# Get started

### Fallow steps

- 1

  ```bash
  $ composer install 
  ```

- 2

  ```bash
  $ mysql -u root
  ```

- 3
  ```bash
  $ CREATE DATABASE db_name;
  ```

- 4
  ```bash
  $ cp .env.example .env
  ```

- 5
  ```bash
  $ php artisan project:install
  ```

- 6
  ```bash
  $ php artisan serve
  ```

- 7 - Copy the email and password of test in console and login
